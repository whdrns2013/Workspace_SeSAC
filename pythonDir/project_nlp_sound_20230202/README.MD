## 1. 프로젝트 소개

<br>

### ⚙️ 주요 기능  

**사투리 TTS - STT**  
* 텍스트를 강원도, 경상도, 전라도, 제주도, 충청도 사투리 음성으로 변환해주는 기능  
* 음성으로 변환된 자료의 Sequence, Mel-Mpectogram 을 출력해주는 기능  

**자기 음성 TTS**  
* 텍스트를 학습된 음성(자신 혹은 특정 타인의 음성 가능)으로 변환해주는 기능
